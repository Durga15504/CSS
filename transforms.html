Transforms

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

1. Translate(x,y)

Moves element left/right, top/bottom using x and y axis.

EX:- transform: translate(50px, 20px);

translate

It moves 50px right side and 20px down. If we give translate(-50px, -20px) then it moves 50px left and 20px top.

2. Scale(x,y)

It increases or decreases the element size.

EX:- transform: scale(1.5,1);

scale

increases width 1.5 times and heigth 1 time.

3. Rotate(angle)

Rotates by using degrees angle in 2D.

EX:- transform: rotate(30deg);

rotate

rotates element by 30deg

4. Skew(x,y)

Rotates by using degrees in 3D or tilts the element along X and Y axis.

EX:- transform: skew(10deg, 20deg)

skew

tilts the element along X-axis in 10deg and Y-axis in 20deg.
