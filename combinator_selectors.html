Combinator Selectors

Child Selector:

It selects only the elements that are direct children (represent with >) of a specific element.

Example HTML:
<div>
    <h1> Child Selector </h1>
    <p> This is a Child Selector </p>
</div>

Here <h1> and <p> are direct children of <div>

Descendant Selector:

Selects all elements that are nested inside another element (at any level) or indirect children (represent with space) of a specific element.

Example HTML:
<div>
    <h1 class=parent> Descendant Selector </h1>
    <p> This is a Descendant Selector </p>
    <div class=child>
          <h2> It is direct child to child class and indirect child to parent class
            
</div>

Adjacent-Sibling Selector:

Selects an element that is the next sibling (represent with +) immediately after another.

Example HTML:

<div>
    <h1> Adjacent-Sibling Selector </h1>
    <p> This is 1st para </p>
    <p> This is 2nd para and is not immediately after the heading. </p>
</div>

Example CSS:

h1 + p {
  color: blue;
  font-weight: bold;
}

Explanation

HERE: h1 + p → Selects the first <p> that comes right after <h1>. This makes only the first paragraph blue and bold.

It does not selects second <p> because it is not immediate sibling.

General-Sibling Selector:

Selects all sibling elements that come after a specific element (not necessarily immediately) and represents with ~.

Example HTML:

<div>
    <h1> General-Sibling Selector </h1>
    <p> This is 1st para </p>
    <p> This is 2nd para and not immediately after the heading. </p>
    <p> This is 3rd para </p>
</div>

Example CSS:

h1 ~ p {
    color: red;
    background-color: yellow;
}

Explanation

HERE: h1 ~ p → It selects all the p elements. The color and background-color applies for 3 p elements i.e., it makes 1st, 2nd and 3rd para red color and yellow background.
