Simple Selectorss

Element Selector:

It selects html elements based on 'tagname'.

Example:
p {
  text-align: center;
  color: red;
}

Here, all < p > elements on the page will be center-aligned, with a red text color. And written in internal or external css.

id selector:

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Example:
#main {
  background-color: yellow;
}

Applies to the element with id="main". And written in internal or external css.

Ex:
<p id="main"> It is a id selector example. </p>

Class Selector:

The class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period (.) character, followed by the class name.

Example:
.center {
  text-align: center;
  color: red;
}

Applies to all elements with class="center". And can applies to multiple elements.

Ex:
<p class="center"> It is a Class selector example. </p>

Grouping Selector:

To group selectors, separate each selector with a comma. It minimizes the code when we want to give same styling for multiple elments we use this selector.

Example:
Instead of writing like this--

h1 {
  text-align: center;
  color: red;
}

h2 {
  text-align: center;
  color: red;
}

p {
  text-align: center;
  color: red;
}

We can minimize the code--

h1, h2, p {
  text-align: center;
  color: red;
}

Universal Selector:

The universal selector (*) selects all HTML elements on the page.

Example:
* {
  margin: 0;
  padding: 0;
}
